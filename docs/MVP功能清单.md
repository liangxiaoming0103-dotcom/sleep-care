# MVP 功能清单

> 依据《智能睡眠环境调控设备 - 软件系统功能需求（实训需求）V1.0》整理，面向实训交付的最小可行产品功能范围。

---

## 1. 用户登录

支持手机号+验证码注册与登录，以及微信一键授权登录（获取昵称、头像）。登录后 Token 自动维护，支持个人信息修改（昵称、头像、性别、出生年份）及符合个保法的账号注销功能。

## 2. 虚拟设备管理

用户可手动输入16位设备序列号或点击"添加演示设备"由后台自动生成虚拟设备并绑定。已绑定设备以列表展示，支持编辑昵称、查看型号/固件版本、解绑操作。虚拟设备默认在线，可手动切换为离线状态用于测试。

## 3. 睡眠报告分期图表

基于后台模拟数据生成器每日自动生成完整版睡眠报告，包含：睡眠阶段曲线（深睡/浅睡/REM/清醒的堆叠条形图或折线图）、心率趋势图（夜间分钟均值）、呼吸频率趋势图，以及异常事件列表（如"呼吸暂停可能"等随机标注）。

## 4. 噪音曲线

睡眠报告中展示环境噪音等级曲线（分钟级 dB SPL 数据），由模拟数据生成器按生理规律自动生成，以折线图形式呈现夜间噪音变化趋势。

## 5. 日周月趋势

支持按日、周、月三种粒度查看历史睡眠报告。首页卡片展示最近一晚睡眠评分（0-100）、总时长、深睡比例、觉醒次数；趋势页展示近7天/近4周/近6月睡眠评分走势图，直观反映睡眠质量变化。

## 6. 个性化作息

用户可设置期望就寝/起床时间（格式 HH:MM）和日出程序时长（5-30分钟），配置助眠音效类型（白噪音/海浪/雨声/篝火）和唤醒声音，以及偏好亮度/音量等级。所有设置存储于云端，支持后续硬件接入时同步。夜间勿扰模式可在设定时段内屏蔽通知推送。

## 7. 医生授权基础

用户输入医生手机号或ID发起授权请求，系统通知医生确认。支持查看已授权医生列表，展示授权剩余天数（默认30天，可选7/15/30/60天），用户可随时撤销授权。授权期间医生可查看用户脱敏后的睡眠报告、睡眠日记、量表测评及打卡记录。

---

*注：OTA固件升级、e-CBTi睡眠训练专区（卫生教育/睡眠日记/专业测评/睡眠限制打卡）、消息通知等模块属于完整版范围，不在本 MVP 清单内。*
